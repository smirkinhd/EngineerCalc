**Инженерный калькулятор с помощью применения JavaScript, HTML, CSS**

- Функции +, -, *, /, %, x^2 работают с n-переменными
- Функции x^2, x^-1, x!, sin, cos, tan, ctg, ln, lg, sqrt работают только с первой переменной
